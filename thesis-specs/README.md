# 论文规范（thesis-specs）

本文件夹汇总哈尔滨工业大学学位论文相关的**官方规范文件**，用于写作与排版时对照、自查。

| 文件 | 说明 |
|------|------|
| `模板.docx` | 哈工大论文**完整 Word 模板**（含样式、页眉页脚、分节、目录域等）。写作或核对版式时的总体参照。 |
| `封面.docx` | 哈工大**官方封面模板**。工具生成时会自动复制此封面到论文开头并替换论文题目（见 `skill/scripts/cover_inject.py`）。 |
| `硕士学位论文格式要求及审查要点.docx` | 哈工大《学位论文格式要求及审查要点》（深圳研究生院），格式自查与送审审查的权威依据。 |

## 与工具的关系

- 工具内置默认 profile 为 **`hit-master-thesis`**（硕士）。生成时自动套用上述规范。
- 本科 / 博士：工具在硕士产物基础上，改写封面、页眉、页脚与后置小节标题中的「硕士 / master」为「本科 / bachelor」或「博士 / doctor」（见 `generate.py --degree`）。
- 真正的格式规则由引擎 profile（`thesis_md2docx/profiles/hit_master_thesis/`）实现；本文件夹的文档是**人工对照与送审**用的原始规范，不参与自动排版。

> 提示：这些文件是规范「原件」，请勿在仓库里直接涂改；如需更新请以学校最新下发版本替换。
