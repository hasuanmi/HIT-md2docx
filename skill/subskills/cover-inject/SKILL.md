# 封面与前置部分版式注入（步骤 3.5）

把《哈工大研究生学位论文书写范例》官方封面模板的版式注入到引擎生成的论文，**仅改动封面与前置部分，不碰引擎、不碰正文样式**。

## 为什么单独成步（而非改引擎）

官方封面是 3 页分节版式（中文封面 / 中文扉页表格 / 英文封面表格 + 分页 + 分节 sectPr），
引擎的 `hit_master_thesis` profile 不生成这种精确版式。逐页照搬只能对 docx 后处理，
写进 profile 属于侵入式改引擎（违反铁律）。故作为标准后处理步骤固化于此。

## 步骤

在步骤 3（fill-template）生成 `output/<论文>.docx` 之后、步骤 4（audit）之前执行：

```bash
python skill/scripts/cover_inject.py output/<论文>.docx \
    --cover-template input/封面.docx \
    --front-matter input/front_matter_hit.md \
    --out output/<论文>.docx
```

也可用绝对/相对路径，或经 `--cn-title` / `--en-title` 直接传题目（优先级最高）。

## 该脚本做了什么

1. **封面整页复制**：读取 `封面.docx`，将其完整封面区（段落 + 表格 + 分页符 + 段落级 sectPr）
   深拷贝到目标 docx 开头，替换目标原有封面区。
2. **只改题目**：把模板里的示例题目（含白色批注箭头）替换为你论文的中/英文题目；白色字批注原样保留。
   题目来源优先级：`--cn-title/--en-title` → `front_matter_hit.md` 的"论文题目："/"英文题目：" → 从现有封面区提取兜底。
3. **目录摘要条目**：在中/英文目录的章节条目之前插入 `摘  要 → I`、`Abstract → II`（罗马数字）。
4. **前置部分行距对齐**：摘要标题段距、摘要正文行距对齐书写范例（中文 1.2 倍、英文 1.5 倍）。

## 健壮性

- **幂等**：每次都基于模板重新注入封面区（先删旧封面区）；目录条目/行距检测已存在则跳过。连跑两次改动数为 0。
- **可选**：`--cover-template` 指向的文件不存在时，跳过封面复制，仅做目录/行距，不报错——流程可无模板运行。
- **不破坏正文**：只动封面区与目录/摘要页，正文（含第七章结论、参考文献、致谢等）原样保留。

## 注意

- 封面占位字段（作者/导师/学科/分类号/答辩日期）由 `front_matter_hit.md` 驱动；模板是官方版式，字段值需你填真实信息后重跑或 Word 内改。
- 若重跑引擎（步骤 3），封面会被引擎版覆盖，需再跑一次本步骤恢复模板版式。
- 中文题目过长时按词边界断 2 行；优先在 "Industrial Technology" 复合词后断（英文题目结构）。
