# 哈尔滨工业大学硕士学位论文 · 格式审查要点（权威提取版）

> 来源：`profiles/hit_master_thesis/format_requirements/硕士学位论文格式要求及审查要点.docx`
> 制定方：哈尔滨工业大学深圳研究生院
> 依据：《哈工大研究生学位论文撰写规范》。本要点重点针对**往届易错点**；如与《撰写规范》不一致，以《撰写规范》为准。
> 用法：用 HIT-md2docx 导出 Word 后，按本清单逐条自查（建议导出前先通读"三、正文撰写"与"五、格式自查要点"）。

---

## 〇、总体硬指标

- **正文字数**：硕士一般 3~5 万字（含图表）；目录之后、结论之前不少于 **35 页**。
- **参考文献**：不少于 **40 篇**；英文文献 ≥ 总数的 **1/2**；近 5 年文献 ≥ 总数的 **1/3**；且须包含近 2 年的文献。

---

## 一、正文前的部分（封面 / 扉页 / 目录 / 摘要 / 关键词）

### 1.1 标题
- [ ] 字数不超过 **25 字**；若有副标题，正标题+副标题不超过 **35 字**。
- [ ] 英文标题：封面及扉页均 **全大写**（如 `RESEARCH ON ...`）。
- [ ] 采用**名词性短语**，不含标点符号。
- [ ] 标题字号 **黑体二号（44 半磅）**；注意中、英文标题之间，以及作者名称之间的行间距。

### 1.2 首页及扉页
- [ ] 所在单位：**深圳研究生院** / `Shenzhen Graduate School`（在职工程硕士按实际所在单位填写）。
- [ ] 学位论文英文用 **`Dissertation`**，不用 `Thesis`。
- [ ] 学位类别：工学硕士用 `Master Degree in Engineering`；工程硕士用 `Master's Degree of Engineering`。
- [ ] 学校代码与密级的**冒号应上下对齐**。

### 1.3 目录
- [ ] **章节编号编到 3 级标题，即 `X.X.X`**（如 `2.3.1`）。这是硬性要求，目录必须显示到"条"一级。
- [ ] 目录中的"英文摘要"条目：可写为 **`Abstract`** 或 **`ABSTRACT`**（原文 1.3(1) 与 5.1(5) 两处并存，两种写法均可接受）。
- [ ] **英文摘要页面标题**：首字母大写 **`Abstract`**（仅首字母大写，非全大写）。见 1.3(3)。

> 注意区分两个语境：
> - **页面上的英文摘要标题**（正文前的英文摘要那一页的大标题）= `Abstract`（首字母大写）。
> - **目录里的英文摘要条目**（目录中的一行）= `Abstract` 或 `ABSTRACT` 均可。
> HIT-md2docx 默认渲染为 `Abstract`（页面标题），符合 1.3(3)。

### 1.4 摘要
- [ ] 为**独立短文**，不用"首先、其次、然后……"或"(1)、(2)、(3)……"等序列词开头。
- [ ] 字数 **500~1000 字**，应主要反映：课题研究背景、研究问题、解决方法及评估结果。
- [ ] 英文摘要做到无病句、无语法错误，尽量用简短句式，而非复合句、长句。

### 1.5 关键词
- [ ] 数量 **3~6 个**。
- [ ] 中文关键词用**分号"；"**分开。
- [ ] 英文关键词用**逗号"，"**分开，且**所有字母小写**（专有名词除外）。

## 二、排版要求

### 2.1 页面设置（版芯）
- [ ] 纸张：**A4** 标准纸（210 mm × 297 mm）。
- [ ] 版芯：宽 **150 mm** × 高 **240 mm（±2 mm）**，含页眉、页码；每页约 **33 行、33 列**。
- [ ] 行距（段前 / 段后）：
  - 章标题：段前 **1 行**、段后 **0.8 行**（上、下行距约 10 mm）。
  - 节标题：段前 / 段后各 **0.5 行**（上、下行距约 7–8 mm）。
  - 条标题：段前 / 段后各 **0.5 行**（上、下行距约 6–7 mm）。
  - 款标题：段前 / 段后各 **0.0 行**（上、下行距约 3–4 mm）。

### 2.2 字体
- [ ] 汉字用**宋体**；**全文的数字**（含公式标号、图号、表号等）、**英文单词**、**字母**、**物理量单位**（如 mm）等用 **Times New Roman**。
- [ ] 矩阵、矢量用**粗斜体**（如矢量 **r**）；单变量用**斜体**（不加粗，如 *x*, *y*）。
- [ ] 上下标：有变量含义的用斜体（如 *xᵢ*）；数字、单词首字母、单位等无变量含义的用正体（如 x1、矩阵转置 Aᵀ 的 T、5°）。
- [ ] 单位符号与数字之间加空格：应写"……长度为 3 cm"，而非"……长度为 3cm"。

### 2.3 字号
- [ ] 正文、公式中用**小 4 号（宋体）**或 **12 pt（Times New Roman）**。
- [ ] 图、表中的文字与题注（图 X-1、表 X-1）用**5 号**或 **10 pt**。

### 2.4 图、表、公式
- [ ] 编号：**图 1-1、表 1-1、(1-1)**（含章号）。
- [ ] 插图之前，文中必须有提示，如"见图 1-1""如图 1-1 所示"；**各级标题后面不可直接插图**。
- [ ] 图、表与其题注**同页**。
- [ ] 分图编号用 **a)、b)** 等，且各分图**同页**。
- [ ] 图表须清楚，图片无模糊毛边；非作者自绘的图片或数据需在恰当位置注明出处（图片在题注中给出，表格中数据以上标形式给出）。
- [ ] 坐标及其单位用括号：`t(s)` 或 `时间(s)`，**不能用斜杠**，如 `t/s`。
- [ ] 当前页放不下一个表时，用**"续表"**：下一页右上角写"表 2-1（续表）"，并**重复表头**（第一行）。
- [ ] 行内公式也需采用公式编辑器编写（如"这里 x=… ，d=… cm"）。
- [ ] 图表上下与文中文字间需留一定位置编排。

### 2.5 页末空白
- [ ] 每页末尾尽量不留空白，通过调整图、文字位置消除（建议每页末尾不超过 3 行）。

## 三、正文撰写

### 3.1 绪论
- [ ] "本文的研究内容"中**不用**"第一章、第二章等"，而用：
  > 本文的主要研究内容如下：
  > (1)……；
  > (2)……；
  > (3)……。

### 3.2 论文主体
- [ ] 正文字数：硕士一般 3~5 万字（含图表）；目录后及结论前不少于 35 页。
- [ ] 章节及各章标题：字数在 **15 字以内**。
- [ ] 除《第一章 绪论》外，**各章最后一节为"本章小结"**（实验方法或材料等为内容的章节可不写）。
- [ ] 正文中所有"（）"应为**中文格式括号**。

### 3.3 结论
- [ ] 分 (1)、(2)、(3) 列出所取得的主要成果，每条尽量用"提出了""建立了"。
- [ ] **2000 字以内**，**不加章标题序号**，**不标注引用文献**。

## 四、参考文献

### 4.1 数量及引用要求
- [ ] 数量：硕士不少于 **40 篇**；英文文献 ≥ 总数 **1/2**；近 5 年文献 ≥ 总数 **1/3**；且包含近 2 年文献。
- [ ] **间接引用**：文献标注在所引内容最后一个字的**右上角**，如"……的研究得到了广泛关注[7]"。单篇 `[1]`、多篇 `[7, 9, 11-13]`；同一文献多次引用标注页码，如 `规定[3]92`。
- [ ] **直接引用**：如"文献[23]研究了……的表达方式时"，**[23] 不用上标**，与正文排齐。

### 4.2 引用范例
**4.2.1 期刊**
- 中：`作者1，作者2，作者3，等. 标题名[J]. 期刊名，年，卷（期）：页码.`
- 英：`Author A, Author B, Author C, et al. 标题名[J]. 期刊名，年，卷（期）：页码.`
- 例：
  - [1] 李晓东，张庆红，叶瑾琳. 气候学研究的若干理论问题[J]. 北京大学学报，1999，35（1）：101-106.
  - [2] Helfat C E, Raubitschek R S. Product Sequencing: Co Evolution of Knowledge, Capabilities and Products[J]. Strategic Management Journal, 2000（21）：961-979.

**4.2.2 会议论文**
- `Author A, Author B, Author C, et al. 标题名[C]//会议名, 出版社地点：出版社，年份：页码。`
- 例：
  - [3] 张佐光，张晓宏，仲伟虹，等. 多相混杂纤维复合材料拉伸行为分析[C]//第九届全国复合材料学术会议论文集（下册）. 北京：世界图书出版公司，1996：410-416.

### 4.3 注意事项
- [ ] 英文作者名：姓（全单词，首字母大写）名（仅首字母，大写，**不加点**）。
- [ ] 论文标题的**每个单词首字母大写**。

## 五、格式自查要点（导出后逐项勾选）

### 5.1 首页、扉页、摘要、目录
- [ ] 首页、扉页的**英文标题所有字母大写**。
- [ ] 学位论文英文翻译用 **`Dissertation`**。
- [ ] 摘要：段落首句无"首先、其次、再次"及"(1)、(2)"。
- [ ] 关键词：中文用"；"；英文用"，"分开，每个单词小写（专有名词除外）。
- [ ] 目录：**`ABSTRACT` 全大写或 `Abstract` 均可**，编号**到 3 级标题（X.X.X）**。

### 5.2 正文
- [ ] 绪论中的"本文研究内容"后不用"第一章……、第二章……"，而用"本文的主要研究内容如下："+ (1)(2)(3)。
- [ ] 页面设置：正文版芯 33 行；宋体 / Times New Roman；小 4 号 / 12 pt。
- [ ] **项标题题序空 4 个半角字符**书写，内容空 4 个半角字符接排。
- [ ] 图、表中的文字 5 号 / 10 pt 字号。
- [ ] 插图之前文中必有提示（"见图 1-1"）；各级标题后不可直接插图。
- [ ] 矩阵、矢量用粗斜体；单变量用斜体（不加粗）；单位（mm、°）、矩阵转置符号 T 等用正体。
- [ ] 缩写术语在正文**第一次出现**必须给出完整形式。
- [ ] 图、表、公式编号按 **X-1** 形式（即含章标题），且正确、连续。
- [ ] 图、表、公式在正文被**正确引用**（常见"希望被引的图"与"实际引用的图"不一致）。
- [ ] "如下图 3-5 所示："应为"如图 3-5 所示。"；"如下表 3-5 所示："应为"如表 3-5 所示。"（去掉冒号）。
- [ ] 横纵坐标单位用括号，如 `t(s)` 或 `时间(s)`，不用斜杠 `t/s`。
- [ ] 同一页不够表示表时，下一页用"表 3-1（续表）"并重复表头。
- [ ] 表格为**三线表**形式。
- [ ] 正文序号用 **(1)、(2)……**；更下一级用 **①、②……** 或 **(a)、(b)** 等。
- [ ] 各页末尾尽量不留空白（尽可能小）。
- [ ] "文献[23]XXX"中的文献号 **[23] 不上标**，与正文排齐。

### 5.3 参考文献
- [ ] 数量 ≥ 40；英文 ≥ 1/2；近 5 年 ≥ 1/3；含近 2 年。
- [ ] 英文作者名：姓（全单词首字母大写），名（仅首字母大写，不加点）。
- [ ] 论文标题每个单词首字母大写。
- [ ] 期刊、会议等引用符合上述范例规则。
- [ ] 用 Word 中"项目（英文 item）"格式，自动对齐每一行。

## 附：最易错点速查（往届高频）
1. **英文摘要标题**写成 `ABSTRACT` → 应为 **`Abstract`**（首字母大写）。
2. **目录只到 2 级** → 必须编到 **3 级（X.X.X）**。
3. **英文关键词**用分号或大写 → 应为**逗号 + 全小写**。
4. **参考文献 [23] 上标** → 直接引用时**不上标**，与正文齐平。
5. **图表编号缺章号**（如 `图1`、`表1`）→ 应为 `图1-1`、`表1-1`。
6. **坐标单位用 `t/s`** → 应为 `t(s)`。
7. **标题后直接插图** → 插图前必须有"见图 X-X"提示。
8. **单位与数字间无空格**（`3cm`）→ 应为 `3 cm`。
9. **扉页英文标题未全大写** → 封面及扉页英文标题须全大写。
10. **结论加章标题序号 / 标引用** → 结论不加序号、不标文献。
