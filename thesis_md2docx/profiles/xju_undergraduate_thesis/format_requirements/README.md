# 新疆大学本科毕业论文格式要求

本目录保存 `xju-undergraduate-thesis` profile 对应的学校格式依据文件，便于维护者对照 profile 实现逻辑、示例输出和学校规范。

文件：

- `附件1：新疆大学本科毕业论文（设计）规范及格式要求.docx`
- `附件1：新疆大学本科毕业论文（设计）规范及格式要求.pdf`
- `附件2：新疆大学本科毕业论文(设计)范例.docx`
- `附件2：新疆大学本科毕业论文(设计)范例.pdf`

维护规则：

- profile 实现代码位于上一级目录。
- 调整页面、标题、正文、图表、公式、参考文献、页眉页脚或目录格式时，应对照本目录文件。
- 本目录文件是格式对照依据，不作为导出模板使用。

规则来源：

| 来源 | 作用 | 典型检查项 |
| --- | --- | --- |
| 附件1规范文件 | 明示规则，作为最高优先级依据 | 页面设置、字体字号、标题层级、正文缩进、参考文献格式 |
| 附件2范例文件 | 反推规范未写清的实际排版细节 | 段前段后、封面字段位置、目录域、页眉页脚、分页效果 |

维护时建议先从附件1拆出硬性规则，再用附件2核对 Word/PDF 的实际视觉效果。如果两者存在不一致，优先按附件1的明确要求实现，并在代码注释、测试或提交说明中写明原因。
